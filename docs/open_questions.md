## Критичные

| # | Вопрос | Статус | Решение |
|---|---|---|---|
| 1 | Возраст ребёнка? | ✅ | 4 года (будет 5). Дошкольник — максимально крупные элементы, математическая защита надёжна ещё ~3 года. |
| 2 | Глубина выборки видео для ленты | ✅ | Случайные 12 каналов из подписок → первая страница uploads (последние видео). При pull-to-refresh — другие 12 каналов. Квота ~14 ед./обновление. |
| 3 | Личный проект или Play Store? | ✅ | Личный проект минимум на год. COPPA/GDPR-K не требуется сейчас. |
| 4 | Оффлайн-режим | ✅ | Только метаданные (лента, история) кэшируются в Room. Видео только при интернете. |

## Технические риски

| # | Риск | Статус | Митигация |
|---|---|---|---|
| T1 | Квота API при поиске (100 ед./запрос) | ✅ | Debounce 800 мс. Поиск используется редко (только родителями). Дневной расход ~390 из 10 000. |
| T2 | YouTubePlayerView и реклама | ⚠️ | Требует тестирования на реальном устройстве с аккаунтом ребёнка. Поведение зависит от типа аккаунта Google. |
| T3 | YouTube API для детских аккаунтов (Family Link) | ✅ | Аккаунт ребёнка — supervised. Решение: добавить email ребёнка как Test User в OAuth consent screen в Google Cloud Console. При первом входе Family Link запросит одноразовое подтверждение родителя. После — работает штатно. |
| T4 | Математическая защита для старших детей | ✅ | Ребёнку 4 года — надёжно ~3 года. Пересмотр при достижении 7 лет. |
| T5 | Отписка через API (50 ед.) | ✅ | Функция нужна — оставить. Редко используется. |
| T6 | Фактическая отписка при блокировке | ✅ | Только локальная блокировка (Room), без вызова subscriptions.delete. |
| T7 | При превышении квоты | ✅ | Показывать full-screen empty state с объяснением. Если есть кэш — показать кэш + баннер. |

## Решения дизайна (зафиксированы)

| # | Решение |
|---|---|
| D1 | Bottom Navigation: 4 таба (Главная, История, Подписки, Поиск) |
| D2 | Настройки — через burger menu в AppBar HomeFragment, за ParentalControl |
| D3 | История — только локальная, не синхронизируется с YouTube |
| D4 | Авто-воспроизведение — отсутствует (по окончании видео ничего не происходит) |
| D5 | Комментарии, лайки, дизлайки — не показываем |
| D6 | Реклама — не показываем (зависит от плеера и типа аккаунта — см. риск T2) |
| D7 | Back gesture в плеере — заблокирован; выход только через кнопку "←" в оверлее |
| D8 | Skeleton вместо spinner для всех списков |
| D9 | Шрифт — Nunito или аналогичный округлый sans-serif |
